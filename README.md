# Menu Flexitarien pour 2 personnes (Lundi à Vendredi)

## Menu Quotidien

**Lundi**
- Déjeuner : Salade quinoa pois chiches, légumes croquants, feta
- Dîner : Ratatouille au four, pois chiches rôtis, riz complet

**Mardi**
- Déjeuner : Wraps houmous, légumes grillés, fromage de chèvre
- Dîner : Pâtes complètes légumes sautés, parmesan

**Mercredi**
- Déjeuner : Buddha bowl lentilles, patate douce rôtie, avocat, graines
- Dîner : Omelette aux épinards, salade verte, pain complet

**Jeudi**
- Déjeuner : Soupe de légumes maison, tartine avocat, oeuf poché
- Dîner : Curry de légumes au lait de coco, riz basmati

**Vendredi**
- Déjeuner : Salade pâtes pesto, tomates cerises, mozzarella
- Dîner : Pizza maison légumes grillés, salade verte

---
