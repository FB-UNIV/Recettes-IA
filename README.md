# Menu Flexitarien pour 2 personnes - 5 jours (Lundi au Vendredi)

## Menu Quotidien

| Jour      | Repas 1 (Déjeuner)                        | Repas 2 (Dîner)                        |
|-----------|------------------------------------------|--------------------------------------|
| Lundi     | Salade quinoa, pois chiches, légumes, feta | Ratatouille au four, pois chiches rôtis, riz complet |
| Mardi     | Wraps houmous, légumes grillés, fromage de chèvre | Pâtes complètes aux légumes sautés, parmesan |
| Mercredi  | Buddha bowl lentilles, patate douce, avocat, graines | Omelette épinards, salade verte, pain complet |
| Jeudi     | Soupe de légumes, tartine avocat, œuf poché | Curry légumes coco, riz basmati |
| Vendredi  | Salade de pâtes pesto, tomates cerises, mozzarella | Pizza maison légumes grillés, salade verte |
